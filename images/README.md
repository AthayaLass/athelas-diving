# Images Folder

This folder is for storing website images.

## Recommended Images

### Hero Section
- Large background image for the hero section (1920x1080px recommended)
- Ocean/diving themed image

### Gallery
- Diving photos
- Underwater scenes
- Equipment photos
- Student photos
- Marine life photos

### About Section
- Team photo
- Diving center photo
- Equipment showcase

### General
- Logo file (if you have one)
- Favicon (16x16 or 32x32px)

## Image Formats
- Use JPG for photos
- Use PNG for logos and graphics with transparency
- Use WebP for optimized web images (if supported)

## Image Optimization
- Compress images before uploading
- Recommended max file size: 500KB per image
- Use appropriate dimensions (don't use 4K images for thumbnails)
